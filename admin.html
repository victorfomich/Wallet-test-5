Импорт адресов с ключами

Инструкция по импорту:

Подготовьте CSV файл со следующими колонками:

  • network - название сети (ton, tron, sol, eth, bnb)
  • address - криптовалютный адрес
  • key_type - тип ключа (private_key, seed_phrase, secret_key)
  • private_key - приватный ключ (для private_key типа)
  • seed_phrase - сид-фраза (для seed_phrase типа)
  • secret_key - секретный ключ (для secret_key типа)
  • seed_length - длина сид-фразы (12 или 24, опционально)
  • name - название сети (опционально)
  • standard - стандарт (опционально)
